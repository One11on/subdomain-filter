# subdomain-filter
This tools filter out all unwanted text and give subdomains only

install

git clone

chmod +x subfilt.sh

usage

-i for input

-o for output

./subfilt.sh -i unarrange-subdomain.txt -o arrange-sub.txt

examplr

asn 232 AAA record sub.doamin.com => sub.domain.com and save => filtered.txt
